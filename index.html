Novelki.pl downloader by Deru

Poradnik instalacji i użytkowania

I. Skrypt do pobierania danych książki z przeglądarki

  1. Najpierw należy zainstalować wtyczkę Tampermonkey - umożliwia ona załadowanie customowego skryptu na stronę novelek.
    • Opera - (link do wtyczki)
    • Firefox - (link do wtyczki)
    • Chrome - (link do wtyczki)
  2. Następnie należy kliknąć przycisk poniżej:
    Zainstaluj skrypt do pobierania nowelek
  3. Powinien wyskoczyć kreator instalacji skryptu Tampermonkey. Wystarczy wcisnąć przycisk "Zainstaluj" i skrypt do pobierania nowelek będzie już gotowy do pracy.
  4. Aby pobrać jakąś książkę, należy przejść na jej podstronę na stronie novelki.pl. Nad tytułem książki powinien znajdować się zielony przycisk "Przeanalizuj nowelkę".
  5. Po kliknięciu "Przeanalizuj nowelkę", skrypt automatycznie otworzy sobie każdy rozdział nowelki i zapisze jego zawartość. Po zapisaniu ostatniego dotępnego rozdziału, skrypt powróci do podstrony nowelki.
  6. Po zakończonym procesie analizowania, obok zielonego przycisku "Przeanalizuj nowelkę" pojawi się drugi, niebieski przycisk z napisem "Pobierz tę nowelkę". Kliknięcie tego przycisku zapisze treść książki w wewnętrznym formacie ".nov.json".
  7. Aby przekonwertować plik ".nov.json" na plik epub, należy skorzystać z narzędzia opisanego poniżej, w punkcie II.

II. Skrypt do konwersji pliku .nov.json na plik .epub

  1. Do działania wymagane jest zainstalowanie środowiska Python 3. Poradnik zakłada, że Python jest juz zainstalowany.
  2. Najpierw należy pobrać i rozpakować archiwum ze skryptem:
    Pobierz skrypt do konwersji nowelek
  3. Następnie należy otworzyć konsolę systemową (cmd, Powershell, dowolny linuxowy terminal) i przejść do folderu, gdzie został rozpakowany skrypt
    cd C:/Users/Przykładowy użytkownik/Pobrane/novelki-converter
  4. Jeżeli to twoje pierwsze uruchomienie skryptu, musisz zainstalować wymagane biblioteki Pythona przez użycie polecenia:
    python -m pip install -r requirements.txt
  5. Aby przekonwertować plik .nov.json na plik .epub, użyj następującej komendy:
    python main.py "C:/Users/Przykładowy użytkownik/Pobrane/nazwa-ksiazki.nov.json"
    (oczywiście zamiast tej przykładowej ścieżki wstaw prawidłową)
  6. Domyślnie skrypt oddziela poszczególne akapity ebooka blokowo (jak na stronie novelki.pl).
    Aby oddzielać akapity wcięciami (jak w polskich książkach), użyj argumentu --paragraph indent
    python main.py --paragraph indent "C:/nazwa-ksiazki.nov.json"
  7. Domyślny rozmiar wcięcia to 1.5em, a rozmiar odstępu akapitów blokowych to 0.56em.
    Możesz zmienić tę wartość używając argumentu --spacing.
    python main.py --paragraph indent --spacing 5cm "C:/nazwa-ksiazki.nov.json"
    (możesz używać dowolnych jednostek obsługiwanych przez CSS)
  8. Pliki .epub które wygeneruje skrypt trafiają do folderu "output" znajdującego się w folderze ze skryptem.
